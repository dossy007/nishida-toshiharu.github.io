                2問目 fizz buzz
                1から100までの数字を出力するプログラミングを書いてください。
                ただし、数字が3の倍数の時は数字の代わりにFizzと出力し、5の倍数の時は数字の代わりにBuzzと出力し、
                3と5の倍数のときは、FizzBuzzと出力すること。
                
    
                    
  • 
                      
    {{list}}
    
                      
      
                            答え：
                            for i in range(1, 101):
                                if i % 3 == 0 and i % 5 == 0:
                                    print("FizzBuzz")
                                elif i % 3 == 0:
                                    print("Fizz")
                                elif i % 5 == 0:
                                    print("Buzz")
                                else:
                                    print(i)
            
                            別解：ここまでできなくて良い(自分もここまでは厳しい..)

                                for i in range(1, end + 1):
                                    print('FizzBuzz'[(4 if i % 3 else 0):(4 if i % 5 else 8)] or i)

                ただし、閏年の時だけ「閏年です」と出力されるようにしてください。

                
    
                    
  • 
                      
    {{list}}
    
                      
      
                    解答：
                    def leap_year(year):
                        if year % 100 == 0 and year % 400 != 0:
                            print(year)
                        elif year % 4 == 0:
                            print('うるう年です')
                        else:
                            print(year)
